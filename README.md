# Coral Island Save Editor

Browser-based save editor for Coral Island.

<a href="https://ziwdon.github.io/coral-island-save-editor/" target="_blank" rel="noopener noreferrer">Try it here</a>

> [!WARNING]
> Always keep a backup of your original save before testing an exported file. There is no guarantee that an edited save will remain internally consistent, accepted by the game, or free from gameplay side effects. Use exported saves at your own risk.

## Editor Sections

The editor is organized into focused sections for common save-editing tasks:

- **Player**: Edit player-specific values such as name, money, and gender.
- **World**: Edit broad save-level values such as season or year.
- **Quests**: Inspect saved quest runtime data and adjust existing quest states, objective statuses, and objective progress.
- **Relationships**: View main relationship characters and set player-specific heart levels.
- **Explorer**: Browse the decoded save structure directly and edit supported primitive values.
